The Good

Kony

End-to-end convenience

Easy to integrate webservices

The Eclipse Studio - Do everything from one place

Ionic

Modern, using familiar technologies

Modular and easily extendable

Open - Huge ecosystem

The Bad

Kony

Closed. Small community

Unit Testing is very difficult

The UI builder! (weak team support)

Ionic

Hybrid apps only

AngularJS is complex for beginners

Few enterprise examples (mostly B2C)

Field Experiences

Kony is not delivering on the enterprise promise

Teamwork and git merging is not well supported with Kony

Ionic looks very promising (PoC only though)
